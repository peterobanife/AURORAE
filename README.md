# AURORAE
Modular interactive hardware platform • Lighting + sensor fusion • Embedded systems • Game logic engine

AURORAE is a modular, reconfigurable hardware platform designed for interactive tabletop experiences, rapid prototyping, and experimental game mechanics. Each board combines lighting, sensors, communication, and embedded logic to create dynamic, multi‑board interactions.

---

## 🚀 Features
- Modular board architecture  
- Multi‑board communication  
- Lighting + sensor integration  
- Event‑driven interaction patterns  
- Embedded game logic engine  
- Expandable system design
